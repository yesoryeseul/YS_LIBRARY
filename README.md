<h1 align="middle">YS Library</h1>

## 프로젝트 소개 📝

카카오 도서 검색 api를 활용한 도서 정보 검색 타입스크립트 토이 프로젝트입니다.

검색창에 원하는 도서 정보를 검색하여 원하는 도서 정보를 제공하며, 개별 도서에 대한 상세 정보(가격, 할인가, 도서 내용, 저자, 출판사 등)를 확인할 수 있습니다.
